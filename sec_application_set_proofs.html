Section 2.2 Application: Set Properties and Equivalences

¶
Definition 2.2.1. Set operations defined via logic.

Let \(A \and B\) be sets.

  1. \(A \subseteq B \iff (x \in A) \to (x \in B)\)
  2. \(x \in A \cup B \iff x \in A \lor x \in B\)
  3. \(x \in A \cap B \iff x \in A \land x \in B\)
  4. \(x \in A \setminus B \iff x \in A \land x \not\in B\)
  5. \(A = B\) if and only if \(A \subseteq B\) and \(B \subseteq A\text{.}\)
Answer/Video
Example 2.2.2.

Prove that if \(S\) is any set:

  1. \(\emptyset \subseteq S\)
  2. \(S \subseteq S\)
Solution

  1. The logical statement we're trying to prove is \((x \in \emptyset) \to (x \in S)\)

    Assume that \(x \in \emptyset\text{.}\) This is false, since nothing is in the empty set. That means that the conditional statement \((x \in \emptyset) \to (x\in S)\) is vacuously true, since the conditional \(F \to p\) is always true. Thus \(\emptyset \subseteq S\text{.}\)

  2. The logical statement we're trying to prove is \((x \in S) \to (x\in S)\)

    Assume that \(x \in S\text{.}\) Then \(x \in S\) is trivially true. Thus \(S \subseteq S\text{.}\)

Example 2.2.3.

Prove the following identities:

  1. \(A \cup \emptyset = A\)
  2. \(A \subseteq (A \cup B)\)
  3. \(A \cap (A \cup B) = A\)
  4. \(\overline{A \cup B} = \overline{A} \cap \overline{B}\)
  5. \(A \cap (A \cup B) = A\)
Answer/Video
Solution

We'll prove the first via an “iff” argument:

\begin{align*} x\in A \cup \emptyset \amp \iff x \in A \lor x\in \emptyset\\ \amp \iff x\in A \lor F\\ \amp \iff x\in A \end{align*}

thus \(A \cup \emptyset = A\text{.}\)

  1. \(A \cup (B \cup C) = (A \cup B) \cup C\)
  2. \(\overline{\overline{A}} = A\)
  3. \(A \cup (A \cap B) = A\)
  4. \(A \cup (B \cap C) = (A \cup B) \cap (A \cup C)\)
  5. \(\overline{A \cap B} = \overline{A} \cup \overline{B}\)
  6. \(A \subseteq B\) if and only if \(\overline{B} \subseteq \overline{A}\)
