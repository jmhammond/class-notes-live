We say that \(A\) is a subset of \(B\text{,}\) and write \(A \subseteq B\) if every element of \(A\) is an element of \(B\text{.}\) If there is an element of \(B\) which is not also in \(A\text{,}\) we say that \(A\) is a proper subset of \(B\) and write \(A \subset B\text{.}\)
