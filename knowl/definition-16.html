Definition 1.3.1.

A mathematical relation, \(\mathrel{R}\text{,}\) between two sets \(A\) and \(B\) is a collection of ordered pairs from the two sets. Symbollically, we write \(\mathrel{R} = \{ (a,b) \mid a\in A \and b\in B\}\text{,}\) or \(\mathrel{R} \subseteq A \times B\text{.}\)

We call set \(A\) the domain and the set \(B\) the codomain.

If \(a\) is related to \(b\) we will write \(a \mathrel{R} b\) or \((a,b) \in \mathrel{R}\text{.}\) If two elements \(c \) and \(d\) are not related, we write \(c \cancel{\mathrel{R}} d\text{.}\)
