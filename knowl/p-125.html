Find the following cardinalities:

  1. \(|A|\) when \(A = \{4,5,6,\ldots,37\}\)
  2. \(|A|\) when \(A = \{x \in \Z \st -2 \le x \le 100\}\)

  3. \(|A \cap B|\) when \(A = \{x \in \N \st x \le 20\}\) and \(B = \{x \in \N \st x \mbox{ is prime} \}\text{;}\) note that 1 isn't prime.
